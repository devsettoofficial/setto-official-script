# Setto Official Bot

Modern WhatsApp bot developed by Setto Official using Baileys with multi device support and modern automation features.

## ✨ Features
- AI ChatGPT
- Auto Response AI
- AI Image Generation
- TikTok Downloader
- Instagram Downloader
- Facebook Downloader
- Twitter/X Downloader
- Sticker Tools
- Group Management
- Multi Device Support

## 🚀 Installation

```bash
git clone https://github.com/username/setto-official-bot.git
cd setto-official-bot
npm install
npm start
